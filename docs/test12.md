1. **从80年代到现在的中国女性体育偶像：女排、邓亚萍、郭晶晶、李娜、张伟丽、谷爱凌，她们身上分别代表了那个时间段的哪些女性品质和投射？**

   从我这个非体育爱好者看来，早期的偶像似乎很少强调性别角色。女排是拼搏精神，邓亚萍郭晶晶是因为强。李娜是脱离举国体制的一次试验，可能还有承认了运动员情绪/个性。傅园慧也是个性解放的一部分，普罗大众发现运动员也是会上网冲浪会开玩笑的普通人。近几年的女性体育偶像，像张伟丽和谷爱凌这种更强调女性力量。她们面对的困境，特别是成名后的困境是更加“女性”的。全红婵被写进族谱，张伟丽被问会不会家暴丈夫......

2. **近年来对于女性运动员外貌的关注增加，为什么？**

   有可能是因为女性更加关注运动员了，但是不知道怎么从专业的角度评论运动员。我们长期不被鼓励运动，而且体育教育也是缺位的，你要一个新手在没有引导的情况下评论比赛成绩训练强度是不现实的。她们只能把关注的着力点放在自己最熟悉的领域：妆容、发型、衣服。我觉得算是某种有心无力的体现。

3. **女性运动员频登时尚杂志，运动员们到底是打开了我们的审美世界还是加固了审美世界的偏见？**

   要看时尚杂志在这个过程中使用了什么权力。时尚行业的权力在于重新定义美或巩固现有美的定义。如果杂志只是让运动员穿和模特一样的衣服、摆出和模特一样的姿势，那只是找一群新的人来为自己定义的美背书。如果杂志像展示大码模特一样，让运动员展示和普通模特不一样的侧面，甚至带有一些让我们“不舒服”的元素，那是在重新定义美。总体来说，我比较偏向后者。我记得奥运会之前，有某个大牌杂志（时尚芭莎？）给谷爱凌拍了一张封面，很多人在下面批评说不好看太粗糙，但是我觉得就很好，因为Vogue在使用自己重新定义美的权力。![谷爱凌vogue](C:\Users\ruizu\OneDrive\桌面\谷爱凌vogue.jpg)

   

4. **谷爱凌作为一个女权缪斯，我们应该如何去看待她？我朋友说谷爱凌的部分发言有些不妥。比如说她说露出头发是为了展现自己是女生，这是不是落入了父权对女性外貌的定义？再比如她说的不想跟uneducated的人解释自己，本身也带有父权社会占据高位者的傲慢？所以谷爱凌所呈现的完美形象本身长期来看，也会是对女性的另一种禁锢和压迫？**

   - 第一个露头发的问题，我觉得确实是迎合了女性外貌的刻板印象，但是她这么做现有情况的局部最优解。露出头发是最直观地表现她是女性的方法之一。如果采用其他方法未必能达到这么一目了然的效果。由于滑雪这个运动女运动员历史上地位并不高，露出头发起到的激励作用可以是巨大的。这个问题就像是冠姓权一样。让孩子跟母亲姓，其实是让孩子跟母亲的父亲姓，而且传承了“姓氏继承财产”这一叙事，但是冠姓权在不平等极为严峻的今天是非常直接实用的提高女性地位的方法。这些短平快的策略会有长期隐患吗？我认为有。但是做什么都有隐患，做了比不做要好。我认为世界上不存在一劳永逸的行动方针，女性议题、女性困境、困境的解决方式都是动态的，时时在变化的。到了以后，等到“露出头发造成女性刻板印象”成为主要议题时，自然会有下一代女性试图解决我们今天的手段留下的隐患。就像几十年前，女人在争取打扮自己的权利，今天女人在追求不打扮自己的权利。
   - 对于uneducated的这个回答，我觉得不是父权带来的傲慢而是阶级带来的傲慢。这个发言我觉得不妥，但是期待一个18岁的天之娇女和普罗众生感同身受是不现实的。她也许还没有等到一个反思傲慢的契机。
   - 以及最后一个“谷的完美形象会是另一个对女性的禁锢和压迫吗”，我认为取决于大众如何解读她。就像宗教经典本身是多面的，只有掌握了释经权的宗教人士才能掌握宗教的形象。谷身上有很多可以挖掘的点，她是运动天才少女、学霸、母系社会传承人、模特、等等等等。鸡娃的家长看到的是天才少女，会加倍逼迫孩子。外貌焦虑少女看到的是头发丝到指甲尖的精致造型，体育爱好者看到的是女性运动的可能性。最后她盖棺定论，起到了什么样的作用，可能取决于这几样人的占比，和他们能发掘出多大的能量来宣传自己的看到的谷爱凌。

5. **这个年代，我们还需要全民偶像吗？**

   - 我的认为现在这个时代既没有能力制造全民偶像，也不需要全民偶像。全民偶像是一代人面对困境时共同渴望的精神力量。但是随着社会的多元化和贫富差距的加大，我们面对的已经是不同的困境了。有的人需要谷爱凌的松弛自信，有的人需要张伟丽重新开始的勇气。就像我们之前讨论谷爱凌的时候说的，即便喜欢的是同一个人，我们喜欢的也可能是不同的侧面。那么，我们还算是“喜欢”同一个偶像吗？
   - 并且，能制造“全民”偶像，是同质化的集体（爱好、经济水平）和有限的信息接收渠道（娱乐活动）作用下的结果。前者刚刚已经论述过了，后者在今天更不成立了。以前娱乐方式都会根据每个人的爱好精准投放。我们很难认识一样的人，也很难喜欢上同样的人了。
